# Telegram bot for Damubala QR

Бот для генерации QR-модалки подписи родителей в стиле Studia.

## Что умеет
- Принимает таблицу детей (`.xlsx/.xls/.csv/.json`) и сохраняет в `data/children.json`.
- Может импортировать локальный файл `База детей.xlsx` командой `/import_local`.
- Ищет детей по части имени и показывает подсказки кнопками.
- После выбора ребенка:
  - входит в Damubala,
  - пробует пароль `Aa123456!`, если не подошел - `Aa123456@` (или значения из `.env`),
  - если Damubala требует обновить пароль, меняет на запасной, как в Studia,
  - сохраняет рабочий пароль обратно в базу,
  - строит QR и отправляет PNG-модалку в Telegram.

## Подготовка
1. Создайте `.env` на основе `.env.example`.
2. Укажите токен Telegram-бота.

## Формат таблицы
Поддерживаются колонки (синонимы):
- `ФИО ребенка` / `ФИО ребёнка` / `childName`
- `ИИН родителя` / `login` / `iin`
- `Пароль` / `password` (можно не указывать)
- `Запасной пароль 1` / `backupPassword1` (можно не указывать)
- `Запасной пароль 2` / `backupPassword2` (можно не указывать)

Если пароли не указаны в таблице, используются значения из `.env`.

## Запуск
```bash
cd "tg bot"
npm install
npm run start
```

## Использование
1. Запустите бота.
2. Выполните `/import_local` (для файла `База детей.xlsx` в папке `tg bot`) или отправьте таблицу в чат с ботом.
3. Напишите имя ребенка (или часть имени).
4. Выберите ребенка из кнопок.
5. Получите PNG с QR-модалкой.
